Now let's start with the coding part.
The code for ACTIONS.js is given below:



 export const ACTIONS = {
    ADD: 'add',
    CHOOSE: 'choose',
    CLEAR: 'clear',
    DELETE: 'delete',
    EVALUATE: 'evaluate'
  
  }
  

This is just a very simple file as we are just exporting the actions we want to do like the: 
ADD action will help us make a String of two or more numbers by combining them in the current state and in the one's place. 
CHOOSE action will help us to choose operations we wanna perform after validating some validations which we will be discussing later.
CLEAR will help us to clear everything.
DELETE will help us to delete a particular number from the one's place same as the backspace button.
EVALUATE will evaluate the operations.

We have imported React and {useState} from the module 'react'. We check have also declared a state variable for input of the tasks. It includes a method setInput to change the value of input.

This component would return a form that would contain an input tag and a button tag, where the input tag will send the info to the  handleChange handler that would save it in the props and same would also be done by the handleSubmit handler that would  prevent the page from refreshing with the code e.preventDefault();.

Then we update the props on submit and generate a random id number for every task using the code:  Math.floor(Math.random() * 10000).


In the return statement we have used a ternary operator whose syntax is : 

(check statement) ? (if condition true):(if condition false).

We will discuss later that why we used this operator in the return statement of the TodoForm.
